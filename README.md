# Formation-tlse-server-node

This project use an in-memory database based on [LokiJS][1]
## Getting Started  
Install NPM dependencies
```
npm install
```
Start the application
```
npm run start
```

## Available Operation  
```
 GET api/books
 GET api/books/:id
```

You can edit data.json to add books

[1]: http://lokijs.org
